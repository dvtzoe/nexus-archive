# Nexus Archive

A node-link-based publishing platform.

## Development

### Setup

```bash
pnpm install
```

### Running

```bash
# Build the web app
pnpm build:web

# Build the repository server
pnpm build:repo


# Start the web app. Make sure to build the web first
pnpm run:web

# Start the repository server. Make sure to build the repo first
pnpm run:repo
```
